# discord_practice_bot
practice discord bot

A basic bot for discord which offers several basic functionalities, 3 of which are aimed at CSGO

The first command is /roll which provides to required values minval and maxval
The command returns a random value through a bot message between the minval and maxval.

![image](https://user-images.githubusercontent.com/111192745/196595068-5467028f-4f3e-4162-9990-f47eed2e9fd1.png)
![image](https://user-images.githubusercontent.com/111192745/196595124-2f02bc9c-bd4b-429e-bae5-d0f1aabe8419.png)

The second command is /randommap which has an optional argument which allows the player to random a map
If the optional value is used they can filter the randommap based on mode

![image](https://user-images.githubusercontent.com/111192745/196595164-a8f47a74-5fc2-4925-8a4c-acf670af582f.png)
![image](https://user-images.githubusercontent.com/111192745/196595211-68692c20-1673-400f-bc5b-da9d8d92d698.png)

The third command is /randomteams which will suggest random T side and CT side teams for participants in the current voicechat of command caller

![image](https://user-images.githubusercontent.com/111192745/196595268-a57c6567-d790-4931-80fa-c997417a3117.png)

A fourth command is planned which will move the aforementioned voicechat participants to a designated voice-chat based on their team.
